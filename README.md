# ArcIris

## Setup
1. ```
   python3 -m venv --upgrade-deps .venv
   ```
1. ```
   source .venv/bin/activate
   ```
1. ```
   pip install -r requirements.txt
   ```
1. ```
   python train.py
   ```
1. ```
   deactivate
   ```
